Initial release of the Reactor JS SDK.

Features:
- `@reactor/client` - Unified client for all Reactor capabilities
- `@reactor/auth` - Authentication with session management, orgs, and API keys
- `@reactor/data` - PostgREST-style query builder for PostgreSQL
- `@reactor/storage` - File storage with bucket management
- `@reactor/functions` - Serverless function invocation
- `@reactor/jobs` - Background job scheduling and management
- `@reactor/sites` - Static site deployment and domain management
- `@reactor/realtime` - Real-time subscriptions (stub)
- `@reactor/shared` - Shared utilities and types
